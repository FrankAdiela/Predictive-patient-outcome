# Project Summary

## Overview

This project was completed as part of my MSc in Applied Artificial Intelligence and Data Analytics.

The objective was to investigate whether machine learning techniques could be used to predict patient outcomes using healthcare data. The project followed an end-to-end analytical workflow, beginning with data preparation and feature engineering before progressing to predictive modelling and model evaluation.

---

## Business Problem

Healthcare organisations generate large volumes of patient data every day. Identifying patients who may be at higher risk of adverse outcomes or hospital readmission can help clinicians make earlier interventions, improve patient care and optimise healthcare resources.

This project explored how machine learning models could support healthcare decision-making by identifying patterns within historical patient data.

---

## Project Objectives

The objectives of this project were to:

- Prepare and clean healthcare data for analysis.
- Assess data quality and handle missing values.
- Perform feature engineering to improve predictive performance.
- Develop predictive machine learning models.
- Compare Logistic Regression and Random Forest models.
- Evaluate model performance using standard classification metrics.

---

## Solution

The project followed an end-to-end machine learning workflow.

Healthcare data was cleaned, transformed and prepared before additional synthetic features were generated to support modelling.

Two supervised machine learning algorithms were developed:

- Logistic Regression
- Random Forest Classifier

The models were evaluated using multiple performance metrics to assess their ability to predict patient outcomes.
